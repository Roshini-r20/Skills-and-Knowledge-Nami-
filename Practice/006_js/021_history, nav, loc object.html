length returns the number of elements in the History list. Usage: history.length;
        
        Methods:
        
        1. back() method, loads previous URL from history list. Usage: history.back();
        
        2. forward() method, loads next URL from history list. Usage: history.forward();
        
        3. go() method, loads previous URL present at the given number from the history list.

        appName: Returns the name of the client.
