1. firstChild: Returns Node object which is the first child of the specified node. Its is equivalent to childNodes[0]. 
        2. lastChild: Returns Node object which is the last child of the specified node. 
        3. nextSibling: returns the Node object of the node that immediately follows the specified node at the same tree level. 
        4. previousSibling: Returns the Node object of the node that the previous node of the specified node at the same tree level.
