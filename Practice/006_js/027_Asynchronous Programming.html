Some of the techniques are:
        1. Callbacks: passed as an argument to another function. Callbacks make sure that a certain function does not execute until another function has already finished execution.
        2. Promises: A Promise is a holder for a result (or an error) that will become available in the future.
        3. Async/Await: "async/await" was introduced to implement asynchronous code with promises that resemble synchronous code.
