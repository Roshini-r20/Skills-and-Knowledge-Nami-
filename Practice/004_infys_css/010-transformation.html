Transformation
            1. translate: function is used to move an element from one position to another position.
            Transform:translate(x,y) x(+ve)-> right, y(+ve) -> down
            2. scale: change(increase/decrease) the size of any element.
            scale(x,y) -> scales the element width(x-value) & height(y value)
            3. rotate(x):  Rotates the element. 
            rotate(x): +ve: clockwise
            4.  skew(x,y) function is used to turn/skew any element to any angle.
